This is a working catalog of agentic systems — the patterns behind them and the demos
that prove them out. There are two ways to use it.

## Learn by concept

The [concept index](/concepts/) is a learning path, from fundamentals to production
concerns:

1. **Foundations** — what an agent actually is, and how it uses tools.
2. **Reasoning & orchestration** — workflows, routing, and multiple agents working together.
3. **Tools & protocols** — MCP, A2A, and agent registries.
4. **Memory & knowledge** — memory and retrieval (RAG).
5. **Interfaces** — browser/computer use, voice, accessibility, multimodal.
6. **Quality** — evaluation and reusable skills.

Each concept links to the demos that demonstrate it.

## See the work

The [gallery](/gallery/) lists every demo, filterable by concept and by access level:

- **Public** — code is on GitHub; jump straight in.
- **Private** — no public code, but a case study shows what it does and how it's built.
- **Local** — lives on a laptop for now; a write-up or video stands in until it's promoted.
